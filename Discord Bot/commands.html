  • Commands(current)


start

Starts your world

This is how you start your world. However if you type RESET after it will delete your world if you have one.

show

Shows your world

This is how you display your world. When you type a number after it will show you the specific layer, otherwize it will show you the surface.

add

makes your world bigger

This is how you add to your world. You have to type the layer number at the end to add a layer. When you add to a layer lower to your lowest layer it will create a new layer.

help

If you have any more questions consult the help command for help.
